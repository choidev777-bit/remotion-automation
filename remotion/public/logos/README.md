# 브랜드 로고 에셋 폴더

이 폴더에 브랜드 로고를 SVG/PNG 파일로 넣으세요.

## 사용법
1. 로고 파일을 이 폴더에 넣음 (예: `claude.svg`, `gpt.png`)
2. 파이프라인이 실행되면, 이 폴더를 자동 스캔하여 가용 로고 목록을 AI 프롬프트에 주입
3. AI가 코드 생성 시, 해당 로고를 `<Img src={staticFile('logos/claude.svg')} />` 형태로 사용

## 규칙
- 파일명은 영문 소문자, 확장자 `.svg` 또는 `.png` 권장
- `_` 로 시작하는 파일은 플레이스홀더로 분류되어 로고 목록에서 제외됨
- 예: `_placeholder.png` (로고 없을 때 기본 이미지)

## 현재 파일 목록
- `_placeholder.png` — 로고가 없는 브랜드용 기본 이미지
